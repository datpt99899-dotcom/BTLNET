BẢN CLEAN QUẢN LÝ CỬA HÀNG MÁY TÍNH

Các thay đổi chính:
- Đổi project/solution/namespace/DataSet sang QuanLyCuaHangMayTinh.
- Chuỗi kết nối dùng Data Source=. ; Initial Catalog=QuanLyBanHangMayTinh.
- Bổ sung helper băm SHA256 cho đăng nhập và thêm role CV04 (kế toán).
- Bổ sung DapperRepository.cs.
- Bổ sung form: frmDonDatHang, frmTraHang, frmThongKeDonHang.
- Menu runtime bổ sung Đơn đặt hàng và Trả hàng.
- Nút Báo cáo mở form biểu đồ LiveCharts.

LƯU Ý:
1. Cần mở solution bằng Visual Studio và Restore NuGet Packages.
2. Do thêm package mới (Dapper, LiveCharts, Guna.UI2), máy của bạn cần Internet hoặc nguồn package nội bộ để restore.
3. CSDL gốc của project vẫn cần chỉnh SQL cho đúng các bảng DonDatHang, DonBanHang, TraHang nếu muốn chạy full nghiệp vụ.
4. Nếu muốn an toàn khi build, có thể tạm bỏ reference Guna.UI2 nếu chưa dùng control nào từ thư viện này.
